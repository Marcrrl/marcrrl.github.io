## About

This is an interactive web application where users can enter special codes to reveal personalized birthday messages. Each code triggers a different response with unique styling and messages in Spanish.

## Available Codes

Try these special codes:
- `happy` or `feliz` - Birthday wishes
- `surprise` or `sorpresa` - Surprise message
- `love` or `amor` - Love message
- `party` or `fiesta` - Party message
- `cake` or `pastel` - Cake message
- `mery` - Special personalized message
